Add your PowerPoint files here.

Example:
- public/portfolio-files/client-deck.pptx

Then attach it to any project card in:
- src/components/Projects.tsx

Example fields inside a project object:
- presentationUrl: "/portfolio-files/client-deck.pptx"
- presentationLabel: "Case Study Deck"

The portfolio will then show a button for that PPTX file on the project card.
